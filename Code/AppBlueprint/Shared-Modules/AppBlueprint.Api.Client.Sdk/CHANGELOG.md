# Changelog

All notable changes to AppBlueprint.Api.Client.Sdk will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- Kiota-generated SDK client for AppBlueprint API
- Type-safe API client with full IntelliSense support
- Authentication support (JWT bearer tokens)
- User management client methods
- Team management client methods
- Organization management client methods
- Subscription management client methods
- Billing client methods
- Request builders for all API endpoints
- Automatic retry policies with exponential backoff
- HTTP middleware pipeline for logging and error handling
- Serialization/deserialization for all DTOs
- Support for pagination and filtering
- Cancellation token support for async operations
- Custom HTTP headers and authentication handlers
- Error handling with typed exceptions
- Configuration options for base URL and timeouts
