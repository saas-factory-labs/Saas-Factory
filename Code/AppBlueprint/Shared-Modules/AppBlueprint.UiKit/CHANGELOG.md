# Changelog

All notable changes to AppBlueprint.UiKit will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- Blazor component library for SaaS applications
- Authentication components (LoginForm, RegisterForm, UserMenu)
- Dashboard components (StatCard, ChartWidget, MetricDisplay)
- Team management components (TeamList, TeamMemberCard, InviteDialog)
- Organization components (OrgSelector, OrgSettings)
- Subscription components (PlanCard, UpgradeDialog, BillingHistory)
- Form components (TextField, SelectField, DatePicker, FileUpload)
- Layout components (AppShell, Sidebar, Topbar, Footer)
- Data display components (DataGrid, Table, Card, Badge)
- Navigation components (Breadcrumb, Tabs, Stepper)
- Feedback components (Toast, Modal, ConfirmDialog, LoadingSpinner)
- Utility components (Avatar, Icon, Tooltip)
- Dark mode support
- Responsive design utilities
- Accessibility features (ARIA labels, keyboard navigation)
- Custom CSS themes
